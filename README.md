# Discord TipBot

A Discord wallet and tipping bot developed in Python for blockchain projects using JSON-RPC.

Originally created for the Dash Platinum blockchain, this project demonstrates blockchain integration using Python, MongoDB, JSON-RPC and the Discord API.

This repository is preserved as part of my software engineering portfolio to showcase backend development, wallet management and blockchain communication.

# Installation

Clone the repository

```bash
git clone https://github.com/FelicioX/discord-blockchain-tipbot.git
```

Install the dependencies

```bash
pip install -r requirements.txt
```

Configure

```
config.json
```

Run the bot

```bash
python tipbotdiscord.py
```

---

# Configuration

The bot requires a local configuration file containing:

- Discord Bot Token
- Blockchain RPC credentials
- MongoDB connection

Sensitive configuration files should **never** be committed to the repository.
